# Admin For YouPin

This is admin application for YouPin open reporting platform.

# Installation

```
$ npm install -g pm2
$ npm install
```

# For Local Development

```
$ npm start
```

```
# Watch change and rebuild continuously on another terminal window.

$ npm run watch
```

# Build

```
$ npm run build
```
